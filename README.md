# Git & GitHub Learning

## About
This repository contains a PDF and notes to help beginners learn Git and GitHub. It explains basic commands, workflows, and concepts for version control.

## Files Included
- About-Git-and-GitHub.pdf : Detailed explanation of Git & GitHub basics.

## Getting Started
1. Download the PDF from this repository.
2. Follow the instructions and examples in the PDF.
3. Practice basic commands in Git Bash or terminal.

## Usage
- Learn basic Git commands like `git init`, `git add`, `git commit`, `git push`.
- Understand GitHub workflows such as cloning, creating branches, and pull requests.
